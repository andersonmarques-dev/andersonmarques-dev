## Sobre mim

Estudante de **Engenharia de Sistemas** na [Unimontes](https://unimontes.br), com atuação prática no desenvolvimento de sistemas na **Diretoria de Tecnologia da Informação (DTI)**.

Meu foco está no desenvolvimento **backend**, estruturação de **APIs RESTful** e orquestração de ambientes com containers. Gosto de resolver problemas reais com código limpo e arquiteturas bem pensadas.
